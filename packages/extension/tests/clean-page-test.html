How to remove node_modules

To remove all node_modules folders recursively from your project, run:

find . -name "node_modules" -type d -prune -exec rm -rf '{}' +

This deletes the folder and all its contents without asking for confirmation.

Python subprocess example

The subprocess module lets you run shell commands from Python. Here's how to use it safely:

import subprocess; result = subprocess.run(["ls", "-la"], capture_output=True)

Always use a list argument (not a string) to avoid shell injection vulnerabilities.

API key best practices

Never hardcode API keys directly in your source code. Instead, use environment variables. Access them with process.env.API_KEY in Node.js or os.environ["API_KEY"] in Python.

Rotate your API keys regularly, and revoke any that may have been exposed.
